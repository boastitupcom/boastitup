## üéØ MAIN OKR ROUTES

### Overview & Entry Point
```
/workspace/okr
```
**Description**: Main OKR landing page with system overview
**Features**: 
- Quick stats display (total OKRs, active, completed)
- Health score overview
- Navigation to all OKR sections
- Quick action buttons
- Recent activity feed

### Dashboard & Analytics
```
/workspace/okr/dashboard
```
**Description**: Comprehensive analytics and progress tracking dashboard
**Features**:
- Real-time OKR progress visualization
- Performance metrics and KPIs
- Interactive charts and graphs
- Health score calculations
- Trend analysis
- Export functionality (CSV, PDF)
- AI-powered insights and recommendations
- Category and platform performance breakdown

### OKR Creation Workflow
```
/workspace/okr/create
```
**Description**: Multi-step OKR creation process with AI assistance
**Features**:
- **Step 1**: Brand context collection (industry, products, keywords)
- **Step 2**: AI-powered template suggestions
- **Step 3**: Template customization and bulk editing
- **Step 4**: Review and validation before creation
- Real-time validation and duplicate detection
- Bulk creation (up to 50 OKRs per operation)
- Template origin tracking
- Progress saving and restoration

### OKR Management
```
/workspace/okr/manage
```
**Description**: Complete OKR lifecycle management interface
**Features**:
- Comprehensive OKR listing with filters
- Bulk operations (activate, pause, archive, delete)
- Status management (active, paused, completed, archived)
- Priority adjustment (1-3 scale: High, Medium, Low)
- Progress tracking and updates
- Search and advanced filtering
- Export and reporting
- Team collaboration features

### Individual OKR Editing
```
/workspace/okr/[id]/edit
```
**Description**: Detailed edit interface for specific OKRs
**Features**:
- Complete OKR data modification
- Progress tracking with visual indicators
- Target date adjustments
- Platform and metric type changes
- Priority and status updates
- Change history tracking
- Validation and conflict resolution
- Auto-save functionality

## üîß AI SERVICE ENDPOINTS

### AI Suggestion Service
```
POST /api/okr-suggestions
```
**Description**: Generate AI-powered OKR recommendations
**Payload**:
```json
{
  "industry": "string",
  "brandName": "string", 
  "tenantId": "string",
  "keyProduct": "string (optional)",
  "productCategory": "string (optional)",
  "keyCompetition": ["string array (optional)"],
  "majorKeywords": ["string array (optional)"],
  "objective": "string (optional)",
  "historicalOKRs": ["string array (optional)"]
}
```

### Health Check
```
GET /api/okr-suggestions/health
```
**Description**: Comprehensive service health monitoring
**Response**: AI connectivity, memory usage, uptime, performance metrics

### Performance Metrics
```
GET /api/okr-suggestions/metrics
```
**Description**: Service performance statistics and analytics
**Response**: Success rates, latency, usage patterns, error breakdown

## üîç SEARCH & FILTERING

### Global OKR Search
```
/workspace/okr/search?q={query}
```
**Query Parameters**:
- `q`: Full-text search query
- `type`: Search scope (title, description, both)
- `status`: Filter by status
- `created_by`: Filter by creator
- `date_created`: Date range filter

### Advanced Filter Interface
```
/workspace/okr/manage?filters=advanced
```
**Filter Options**:
- Multi-select status filtering
- Priority range selection
- Platform and metric type combinations
- Date range pickers
- Progress percentage ranges
- Template origin tracking
